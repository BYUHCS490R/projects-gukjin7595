The form tag has an "action" attribute. What does it do?

This action specifies where the form data is sent

The form tag has a "method" attribute. What does it do?

This method tag specifies how the data is sent to the server

What is the difference between a "POST" and a "GET"?

Post sends a data in the request body, no size limit and it's more secure. It's used for saving data.

Get sends data in the URL, limited length and less secure than POST. It's used for retrieving data.
