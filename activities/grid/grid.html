describe the parent-child relationship in HTML and why it becomes essential when using flex and grid.

In HTML, the parent element contains child elements. When we use flex or grid, the parent become the container and only its direct children elements become flex or grid.

What are the two main differences between CSS Grid and CSS Flex?

Grid is two dimensional. it controls rows and columns. Flex is one dimensional. It controls row or column.
